
---

## 💻 Como Executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

---

## ⚠️ Observações

- O sistema não possui backend
- Os dados de usuário são armazenados apenas no navegador (LocalStorage)
- Ao limpar o navegador, os dados são perdidos

---

## 📱 Responsividade

O sistema possui adaptação básica para dispositivos móveis utilizando **media queries**.
